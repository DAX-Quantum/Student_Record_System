# 🎓 Student Record Management System

A command-line Student Record Management System built using Bash scripting on Linux.

## ✨ Features

- ➕ Add Student
- 📋 View Students
- 🔍 Search Student
- ✏️ Update Student
- 🗑️ Delete Student
- ✅ Duplicate Roll Number Validation
- 📄 File-based Database
- 🎨 Formatted Table Output

## 🛠 Technologies Used

- Bash Shell
- Linux
- grep
- printf
- while loops
- case statements
- File Handling

## 📂 Project Structure

```
Student-Record-System/
├── student.sh
├── README.md
├── .gitignore
├── screenshots/
└── students.txt
```

## 🚀 How to Run

```bash
chmod +x student.sh
./student.sh
```
